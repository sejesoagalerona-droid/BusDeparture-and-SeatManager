## Project Description
The Bus Departure and Seat Manager System is a Java console-based application designed to simulate the management of bus departures and passenger seat bookings in a local transport environment. The system allows users to manage buses, track available seats, and handle passenger bookings efficiently.
This project demonstrates key concepts of object-oriented programming such as encapsulation, inheritance, and polymorphism.

## Features
### core features
-**Add Buses**- Register and manage new buses departures
-**Schedule Trips**- Set depature times and destinations
-**Book seats**- Reserve seats using pasanger details
-**cancel Booking**-Remove or update existing reservations 
-**view seats**- view avaliable and booked seats 
-**Manage multiple buses**- Handle serveral buses and trips at once

### Technical Features 
-Abstract 'Bus' class
-'LocalBus' and 'LongDistanceBus' inheritance 
-Polymorphic methods(e.g seat handling)
-Encapsulated seat and bookings management 
-Use of 'ArrayList' for storing buses and passangers
-Input validation and exception handling 
-Input modular package structure(app,model,service,util)
