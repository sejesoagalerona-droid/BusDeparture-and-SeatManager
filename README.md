# BusDeparture-and-SeatManager
CSI142 GROUP 23 ASSIGNMENT


## Project Description
The Bus Departure and Seat Manager System is a Java console-based application designed to simulate the management of bus departures and passenger seat bookings in a local transport environment. The system allows users to manage buses, track available seats, and handle passenger bookings efficiently.
This project demonstrates key concepts of object-oriented programming such as encapsulation, inheritance, and polymorphism.
